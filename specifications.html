Specifications

User Stories

  • As a User:
    • I want a mobile app so that I can access the information I need on-the-go
    • I want a display that is visible in low-lighting so I can see the information in dark environments
    • I want to be able to access activation cards for different trauma centers so that I can accurately route patients
    • I want quick information on a partiular trauma center so that I don't have to waste important time searching
    • I want a simple, intuitive application so that I can get started without "learning" the application
    • I potentially want trauma center recommendations based on GPS so that I can find the nearest locations without manually searching
  • As an Admin:
    • I want to update content on the app without knowing how to code so I can present paramedics with the most relevant information

Requirements

Functional
  • Definite:
    • Application must provide accurate information on activation code criteria for all trauma centers it records
    • Application must provide accurate information on the services available at different trauma centers
    • Application must provide contact information for each trauma center
  • Perhaps:
    • Application may provide information on destination protocols to trauma employees navigate their transport
    • Application may show closest trauma centers to the user's location via GPS assuming internet connection is available
    • Users may search directly for best-fit activation cards by injury descriptions or trauma center name
  • Improbable:
    • Application will notify a selected trauma center automatically of an incoming patient's activation code and the medical justification for that activation code
Non-functional
