Google Account

The Google account, comp523.unc.smarts@gmail.com, allows the user access to the Gmail for the account, the Google Drive, where certain files (like credentials.txt and api_key.txt) exist that are necessary for running the application locally. In the Google Drive there also exist other valuable files, like the signing key/certificate for the Google Play application, which is necessary to update the application found in the Google Play Store.

The Google account also allows the user access to the Google Play Developer Console, where the user can view the specifications for the application in the Play Store and make changes as necessary.

GitHub

GitHub is where the code for the application is stored and updated. The README.md file found there contains more information on the application. It is to the GitHub that heroku connects to run the Server, to which both the Web Portal and Mobile App connect.

Heroku

Heroku is a cloud-based Platform as a Service (PaaS) on which the backend server and database of the application runs. It is through Heroku that the server connects with the MongoDB database using mLab MongoDB. To view the mLab interface for the given application, one need only login to Heroku, click on the link to the given application (named comp523-statt-web-portal), click on the resources header, and finally click on the link "mLab MongoDB."

Expo

Expo is the software used for modeling the mobile application and for creating and updating the mobile application in the Google Play and iOS stores.

The login information and ownership of all of these accounts are handed over to the client for their possession and management.
